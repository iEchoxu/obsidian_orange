>通过 css 修改 obsidian 图片显示方式,让多个图片并列显示在一行.
---
## 用 obsidian 图片裁剪功能实现并列显示图片
``` css
/*不使用css也可实现图片并列显示*/
![[home1.b1c718b9.png|200]]  ![[htext.png|200]] ![[home1.b1c718b9.png|200]]

/* 图片并列显示时设置图片之间的间隔 */
.view-content .markdown-preview-view img  {
  padding: 20px 20px;
}
```

- `![[testc.jpg]]`将按默认图片显示方式来显示图片
